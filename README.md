# FoodScan

FoodScan is a smart food analysis application that helps users identify food details such as allergens, ingredients, or nutritional information by scanning products.

## Features

- Scan food items using camera or barcode
- Detect common allergens (nuts, dairy, gluten, etc.)
- Display ingredient or nutrition data
- Simple and user-friendly interface

## How It Works

FoodScan uses a combination of:
- Barcode scanning / image input
- A food database API
